# Visual Studio Code Landing Page Clone

This project is a **frontend clone of the Visual Studio Code official landing page**, built using **HTML and CSS**.  
The goal of this project is to practice **layout structuring, flexbox, spacing, and responsive design concepts** while recreating a real-world website UI.
